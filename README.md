## Dependencies
  ##### carousel_slider
  - for home screen UI
  - version : 4.2.1
  ##### flutter_linkify
  - for using clickable link
  - version : 6.0.0
  ##### url_launcher
  - for executing link
  - version : 6.2.6
  ##### cloud_firestore:
  - for using firestore that is database of firebase 
  - version : 4.17.3
  ##### firebase_core: 
  - for using firestore that is database of firebase
  - version : 2.31.0
